Beam Analysis Toolkit

A Python-based structural analysis tool for modelling statistically determinate beams under various loading conditions. This project computes support reactions and generates Shear Force Diagrams (SFD) and Bending Moment Diagrams (BMD) for engineering analysis and visualisation. It allows users to define beam geometry, apply different load types, and automatically compute structural responses.

Features: Support for multiple load types: Point loads Uniformly distributed loads (UDL) Linearly varying loads

Automatic calculation of: Support reactions Shear force distribution Bending moment distribution

Generation of: Shear Force Diagrams (SFD) Bending Moment Diagrams (BMD)

Technologies Used: Python NumPy Matplotlib Dataclasses

How It Works: Define the beam length and support conditions Add loads (point loads, UDLs, etc.) Compute reactions using static equilibrium Generate shear force and bending moment distributions Visualise results using plotted diagrams
